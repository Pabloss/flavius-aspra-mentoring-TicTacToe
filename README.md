# flavius-aspra-mentoring-TicTacToe

Requirements
============

* Game has
    * a Board
        * with 9 tiles
            * 3 x 3
    * Two users:
        * User X
        * User O
* Rules:
    * User X starts
    * Users play one after the other
    * The one wins who sign tiles in (before the other)
        * diagonal
        * row
        * column
